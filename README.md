# 🚀 Go Raksha – Smart Livestock Safety App

## 📌 Overview

**Go Raksha** (Go Safety) is a mobile application built to support farmers during livestock health emergencies such as Lumpy Skin Disease. The app is designed with a **simple, fast, and offline-friendly interface in Kannada**, making it accessible for rural users.

---

## 🎯 Objective

To provide farmers with **quick access to veterinary help, disease awareness, and emergency tools** to protect livestock and reduce losses.

---

## ✨ Key Features

* 📍 **Nearby Veterinary Help** – Find vets quickly in emergency
* 🚨 **Emergency SOS Alert** – Send instant alerts for help
* 📷 **AI-Based Disease Detection** – Detect livestock issues using camera
* 📚 **Disease Awareness (Kannada)** – Easy-to-understand guidance
* 🔊 **Voice Assistance** – Helps non-literate users
* 📶 **Offline Mode** – Works even without internet

---

## 🛠️ Tech Stack

* **Frontend:** Flutter
* **Backend:** Firebase
* **AI/ML:** Python + OpenCV
* **Database:** Cloud Firestore

---

## 📱 Target Users

* Farmers 🧑‍🌾
* Veterinary Assistants 🏥
* Rural Communities 🌾

---

## 💡 Innovation

* Focused on **rural usability (Kannada UI)**
* Combines **AI + Mobile + Offline support**
* Designed for **real-world emergency scenarios**

---

## 🚀 Future Scope
